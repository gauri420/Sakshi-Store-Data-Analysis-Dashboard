Sakshi Store Data Analysis Dashboard
1. Project Objective
The primary objective of this project is to analyze the annual sales data of Sakshi Store for the year 2022. By identifying trends, customer preferences, and geographic performance, the analysis aims to provide data-driven insights that can help the store improve sales strategies and customer engagement for the upcoming year.

2. Dataset Used
The analysis is based on a comprehensive sales dataset (Excel/CSV) containing over 31,000 transactions. Key fields include:

Order Details: Order ID, Date, Status, Channel (Amazon, Flipkart, etc.).

Customer Demographics: Gender, Age, Age Group (Teenager, Adult, Senior).

Product Information: Category (Kurta, Set, Western Dress), Size, Quantity, and Amount.

Location: Shipping City and State.

3. Files Included
📁 Excel Project 1.xlsx - Main analysis file with Dashboard.

📁 Cleaned Dataset.csv - Processed data used for analysis.

4. Business Questions & KPIs
Sales Trends: What is the monthly sales and order distribution? (KPI: Total Sales, Total Orders)

Customer Profile: Who is the primary customer? (KPI: Sales by Gender & Age Group)

Regional Performance: Which states are contributing the most? (KPI: Top 5 States by Revenue)

Channel Efficiency: Which platform drives the most orders? (KPI: Order Count per Channel)

Operational Health: What is the ratio of delivered vs. returned orders? (KPI: Order Status Percentage)

5. Process
Data Cleaning: Used Excel to remove duplicates, handle null values, and standardize column formats.

Data Processing: Created new columns such as "Age Group" and "Month" using Excel formulas.

Exploratory Data Analysis (EDA): Built Pivot Tables to summarize complex data.

Data Visualization: Designed an interactive dashboard using Pivot Charts and Slicers.

7. Project Insights
April Peak: April was the most successful month in terms of sales.

Women-Centric: Approximately 64% of the total revenue is generated by female customers.

Adult Dominance: The "Adult" age group (30-49 years) is the most profitable segment.

Geographic Leaders: Maharashtra, Karnataka, and Uttar Pradesh are the top states.

8. Final Conclusion
To maximize growth, Sakshi Store should focus its marketing efforts on Women in the 30-49 age bracket living in Maharashtra and Karnataka. Promotions should be prioritized ahead of the April peak season.
